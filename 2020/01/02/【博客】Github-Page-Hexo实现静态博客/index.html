功能简介

（1）发布/删除文章

发布/删除只需要在hexo根目录中的source/_post/文件夹下创建/删除 md文件，并输入 hexo d -g 就可以了。

发布也可以用 hexo new "文章名" 来创建。

（2）写文章时插入图片

在本地写makedown的时候能看到图片，但是部署之后，在网页上却看不到
